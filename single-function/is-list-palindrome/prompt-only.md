## Prompt

Given a linked list of integers, determine whether or not it's a palindrome.

Reminder: unless otherwise specified, linked list problems are always _singly_ linked list problems.

## Time to solve

30 minutes.


## Examples

### Example #1

* Input: `0 -> 1 -> 0`
* Output: `true`

### Example #2

* Input: `1 -> 2 -> 2 -> 3`
* Output: `false`
